### 특수한 전제
#### 연장신청
- 11시 연장신청은 매일 오후 5시 30분부터 8시 30분까지
- 12시 연장신청은 매일 오후 5시 30분부터 10시까지
#### 외출신청
- 확실히 정해진 조건 없음
#### 잔류신청
- 매주 일요일 오후 8시 30분부터 목요일 오후 10시까지
#### 설문조사
- 한번 업로드한 설문조사는 수정 불가능
